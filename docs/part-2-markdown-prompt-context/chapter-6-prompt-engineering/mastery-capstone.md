# Prompt Engineering Mastery Capstone

## Overview

This capstone project tests your prompt engineering skills by having you write specifications for a complete feature.

## The Challenge

Create a comprehensive specification for a **User Settings Page** that includes:

1. Profile information editing
2. Notification preferences
3. Security settings (password change, 2FA)
4. Theme preferences (light/dark mode)
5. Account deletion

## Requirements

Your specification should include:

### 1. Feature Overview
- Clear description of the feature
- User stories or use cases

### 2. Technical Requirements
- Data models
- API endpoints
- Frontend components

### 3. Business Logic
- Validation rules
- State transitions
- Error handling

### 4. Constraints
- Technical constraints
- UX constraints
- Security requirements

### 5. Examples
- Input/output examples
- UI mockup descriptions
- Error message examples

## Evaluation Criteria

| Aspect | Weight |
|--------|--------|
| Clarity | 25% |
| Completeness | 25% |
| Specificity | 25% |
| Practicality | 25% |

## Self-Assessment

After completing your specification:

1. Give it to an AI and ask it to implement
2. Review the generated code
3. Note any misunderstandings
4. Refine your specification
5. Repeat until satisfied
